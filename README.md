# Online-Shopping-System-PL-SQL
Online Shopping System using PL/SQL with tables, loops, cursors and exception handling
## Project Description
This project demonstrates PL/SQL programming by implementing a simple online shopping system.

## Features
- Created PRODUCTS and ORDERS tables
- Implemented PL/SQL blocks for operations
- Used loops and cursors
- Applied exception handling

## PL/SQL Concepts Used
- Variables and conditions
- Loops and cursors
- Exception handling
- DML operations

## Learning Outcome
- Hands-on experience in PL/SQL
- Improved logic building skills
- Understanding of real-time operations
